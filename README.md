## 简介

主要用于一加及其他机型一键 ROOT。支持 Magisk、Alpha、kitsune、APatch、KernelSU、Kernel-Next、SukiSU Ultra、ReSukiSU 等目前所有主流 ROOT 方式。

## 功能

- 一键 ROOT
- Bootloader 解锁
- 深度测试工具
- 系统维护与优化
