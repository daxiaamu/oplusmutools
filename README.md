## 简介

一加全能工具箱是一款专为一加手机设计的 Windows 端综合工具，提供刷机、解锁、ROOT 等一站式服务。

## 功能

- 一键解锁 Bootloader
- 刷入 Recovery / 刷机
- ROOT 权限管理
- 深度测试工具
- 系统维护与优化

## 下载

前往 [Releases](https://github.com/daxiaamu/oplusmutools/releases) 下载最新版本。

## 使用说明

1. 下载 `一加全能工具箱.exe`
2. 以管理员身份运行
3. 连接一加手机（需开启 USB 调试）
4. 按提示操作即可
